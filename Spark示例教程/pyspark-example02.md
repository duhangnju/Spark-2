# Example02: 使用PySpark进行数据探索

## Step01: 加载csv文件
正确的加载数据，是所有工作的基础。

```python
res = spark.read.csv("/data/jm/practice/jm_daoall2_tags3/000000_0_format/result/*")
```

加载数据的时候，可以对各个字段的类型进行定义，也可以指定是否有header等。

## Step02: 查看基础的数据结构
加载好数据之后，首先需要理解数据结构，主要是字段及其类型。
